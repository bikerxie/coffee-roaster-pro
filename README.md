## 🎯 项目愿景

Coffee Roaster Pro 是一套**软硬一体化**的智能咖啡烘焙解决方案。通过多传感器数据融合、AI 实时分析和精确执行控制，实现从生豆到熟豆的**全自动化无人值守烘焙**。

### 核心创新

| 特性 | 传统烘焙机 | Coffee Roaster Pro |
|------|----------|-------------------|
| 温度监测 | 单点温度 | **双通道 + RoR 实时计算** |
| 烘焙度判断 | 经验/取样 | **色度传感器实时监测** |
| 爆点检测 | 人耳监听 | **AI 智能麦克风自动识别** |
| 下豆控制 | 手动操作 | **自动精准下豆** |
| 烘焙一致性 | 依赖经验 | **AI 曲线预测 + 自动调节** |

## 📱 功能特性

### 🔥 核心功能

#### 1. 多传感器实时监控
- **🌡️ 温度监测** - 豆温 + 风温双通道，每秒 2 次采样
- **🎨 色度识别** - 实时监测豆子表面颜色变化，精确判断烘焙度
- **🎤 音频检测** - AI 智能麦克风，自动识别一爆/二爆
- **⚡ 升温速率** - RoR (Rate of Rise) 实时计算

#### 2. 全自动智能烘焙
- **🤖 一键启动** - 选择预设曲线，全程自动完成
- **🧠 阶段识别** - 自动识别脱水期、梅纳期、一爆期、发展期
- **⚙️ 自动调节** - 根据阶段自动调节火力、转速、风量
- **🫘 自动下豆** - 到达目标烘焙度自动下豆

#### 3. AI 智能预测
- **⏱️ 完成时间预测** - 基于当前 RoR 预测剩余时间
- **🎯 最佳下豆时机** - 综合温度 + 色度 + 时间推荐下豆点
- **📊 异常检测** - 升温过快/过慢自动预警
- **🔮 曲线预测** - 预测未来 60 秒温度走势

#### 4. 专业级曲线控制
- **📈 双轴图表** - 温度(左) + RoR(右) 同时显示
- **🎚️ 精准控制** - 火力 0-100%，转速 0-100%，风量 0-100%
- **🏷️ 事件标记** - 一键标记 FC/SC/Drop，支持自定义标记
- **🔄 曲线对比** - 当前曲线 vs 参考曲线对比显示

#### 5. 数据管理
- **💾 本地存储** - SQLite 数据库，离线可用
- **📤 数据导出** - 支持 Artisan 格式导出
- **📊 烘焙报告** - 自动生成烘焙报告（温度曲线、发展率、失重率）
- **☁️ 云端同步** - 可选云端备份（开发中）

#### 1. 温度传感器系统


**技术规格：**
- **豆温传感器**: K-type 热电偶，响应时间 < 0.5s，精度 ±1°C
- **风温传感器**: K-type 热电偶，安装在热风出口
- **采样率**: 2Hz (每秒 2 次)
- **温度范围**: 0-400°C

## 🧠 AI 算法架构

### 多传感器融合算法

```python
# 伪代码示例
class SensorFusion:
    def analyze_stage(self, temp, color, audio, time):
        # 权重配置
        temp_weight = 0.4
        color_weight = 0.3
        audio_weight = 0.3
        
        # 各传感器判断
        temp_stage = self.from_temperature(temp)
        color_stage = self.from_color(color)
        audio_stage = self.from_audio(audio)
        
        # 加权融合
        final_stage = weighted_vote(
            [temp_stage, color_stage, audio_stage],
            [temp_weight, color_weight, audio_weight]
        )
        
        return final_stage
```

## 🚀 快速开始

### 环境要求

**软件:**
- Flutter SDK 3.27+
- Dart 3.0+
- Android Studio / Xcode

**硬件:**
- 支持蓝牙 LE 5.0 或 WiFi 的智能烘焙机
- 或配合我们的硬件方案 DIY

### 安装步骤

```bash
# 1. 克隆仓库
git clone https://github.com/bikerxie/coffee-roaster-pro.git
cd coffee-roaster-pro

# 2. 安装依赖
flutter pub get

# 3. 配置硬件连接（修改 lib/config/hardware_config.dart）

# 4. 运行应用
flutter run
```

### 硬件配置示例

```dart
// lib/config/hardware_config.dart
class HardwareConfig {
  // 温度传感器

  // 执行器

}
```
