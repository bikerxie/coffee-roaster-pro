# ☕ 咖啡烘焙机控制应用 - Figma 设计文档

## 📱 主要界面描述

### 1. 主控制界面 (Dashboard)
![主界面示意图]

**布局描述：**
- 顶部状态栏：设备连接状态（左）、当前时间（中）、WiFi/蓝牙信号（右）
- 中央温控区：巨大的圆形温度显示，外圈为实时温度进度环
- 烘焙曲线图：位于温度显示下方，实时显示温度-时间曲线
- 底部操作区：三个超大按钮（开始/暂停/停止）
- 侧边数据栏：风量(RPM)、火力(%)、风压(kPa)

**关键元素：**
- 温度数字：使用等宽字体，显示当前豆温/排温/环境温
- 进度环：根据烘焙阶段变色（脱水→黄色，梅纳→橙色，发展→棕色）
- 操作按钮：带触觉反馈的大按钮，支持手套触摸

### 2. 设备连接界面
![连接界面示意图]

**布局描述：**
- 中央：咖啡机图标动画（搜索/已连接状态）
- 设备列表：可滚动的发现设备卡片
- 连接按钮：醒目的蓝色连接按钮

### 3. 历史记录界面
![历史界面示意图]

**布局描述：**
- 顶部筛选栏：日期范围、豆种、烘焙度筛选
- 卡片列表：每次烘焙的摘要卡片
  - 日期时间
  - 豆种/产地
  - 最终温度/时间
  - 烘焙度色卡
  - 评分/备注

### 4. 设置界面
![设置界面示意图]

**布局描述：**
- 分组设置项：
  - 设备设置（PID参数、传感器校准）
  - 烘焙预设（保存的曲线模板）
  - 通知设置
  - 系统信息

## 🎨 颜色规范

### 主色调
| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| **深空黑** | `#0D0D0D` | 主背景色 |
| **炭灰** | `#1A1A1A` | 卡片背景、次级背景 |
| **石墨灰** | `#2D2D2D` | 按钮默认状态、分割线 |
| **钢铁灰** | `#404040` | 边框、禁用状态 |

### 强调色
| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| **熔岩橙** | `#FF6B35` | 主要操作按钮、温度高值警示 |
| **热力红** | `#FF3B30` | 停止按钮、紧急操作、危险温度 |
| **活力绿** | `#34C759` | 开始按钮、安全状态、已连接 |
| **科技蓝** | `#007AFF` | 连接按钮、链接、信息提示 |
| **琥珀黄** | `#FFCC00` | 警告、预热状态、脱水阶段 |

### 温度指示色阶
| 温度范围 | 颜色 | 说明 |
|---------|------|------|
| < 150°C | `#4CAF50` 翠绿 | 预热/脱水阶段 |
| 150-180°C | `#FFC107` 琥珀 | 梅纳反应初期 |
| 180-200°C | `#FF9800` 橙 | 梅纳反应中期 |
| 200-220°C | `#FF6B35` 深橙 | 发展阶段 |
| > 220°C | `#FF3B30` 红 | 一爆后/发展期 |

### 文字颜色
| 用途 | 色值 | 透明度 |
|------|------|--------|
| 主要文字 | `#FFFFFF` | 100% |
| 次要文字 | `#FFFFFF` | 70% |
| 占位符/提示 | `#FFFFFF` | 50% |
| 禁用文字 | `#FFFFFF` | 30% |

## 🔤 字体规范

### 字体家族
```css
--font-display: 'SF Pro Display', -apple-system, sans-serif;
--font-mono: 'SF Mono', 'Courier New', monospace;
--font-condensed: 'SF Pro Display Condensed', sans-serif;
```

### 字号规范

| 用途 | 字体 | 字号 | 字重 | 说明 |
|------|------|------|------|------|
| **超大温度** | SF Mono | 96px | Bold | 主温度显示 |
| **大标题** | SF Pro Display | 48px | Bold | 页面标题 |
| **中标题** | SF Pro Display | 32px | Semibold | 卡片标题 |
| **小标题** | SF Pro Display | 24px | Medium | 分区标题 |
| **正文** | SF Pro Display | 18px | Regular | 常规内容 |
| **辅助文字** | SF Pro Display | 14px | Regular | 标签、说明 |
| **数据数字** | SF Mono | 48px | Bold | 实时数据 |
| **小数据** | SF Mono | 24px | Medium | 次级数据 |

### 特殊字体处理
- **温度数字**：等宽字体确保数字对齐，防止跳动
- **时间显示**：使用等宽字体，格式 `MM:SS`
- **百分比**：数字与%符号间距 `tracking: -0.02em`

## 🧩 组件规范

### 1. 主操作按钮 (Action Button)
```
尺寸：120px × 120px (圆形)
圆角：完全圆形
背景：渐变 (底部较亮增加立体感)
阴影：0 4px 12px rgba(0,0,0,0.4)

状态样式：
- 默认：#2D2D2D 背景 + 内发光
- 按下：#1A1A1A 背景 + 内阴影
- 禁用：#404040 背景 + 50% 透明度

图标尺寸：48px
图标颜色：根据功能 (绿/红/橙)
```

### 2. 数据卡片 (Data Card)
```
尺寸：flexible, 最小 160px × 120px
背景：#1A1A1A
圆角：16px
内边距：20px
边框：1px solid #2D2D2D

内容结构：
┌─────────────────┐
│  标签 (14px)     │
│                 │
│  数值 (48px)     │
│  单位 (14px)     │
└─────────────────┘
```

### 3. 温度环形指示器 (Temperature Ring)
```
尺寸：280px × 280px
圆环粗细：16px
背景轨道：#2D2D2D

进度指示：
- 进度条：渐变色 (根据温度范围变化)
- 刻度标记：每30°一个标记点
- 当前值指示器：带发光效果的小圆点

中心内容：
- 主温度：96px 等宽字体
- 目标温度：24px 次要文字
```

### 4. 曲线图表 (Roast Curve)
```
尺寸：100%宽 × 200px高
背景：透明
网格线：#404040, 虚线, 1px

曲线样式：
- 豆温曲线：#FF6B35, 3px 粗实线
- 排温曲线：#34C759, 2px 粗虚线
- 目标曲线：#404040, 2px 细实线

坐标轴：
- X轴：时间 (0-15分钟)
- Y轴：温度 (0-300°C)
- 标签：14px, 50%透明度

交互：
- 触摸显示数据点详情
- 双指缩放时间范围
```

### 5. 开关控件 (Toggle)
```
尺寸：60px × 32px
轨道圆角：16px

状态：
- 关闭：#404040 背景
- 开启：#34C759 背景

滑块：
- 尺寸：28px 圆形
- 颜色：#FFFFFF
- 阴影：0 2px 4px rgba(0,0,0,0.3)

切换动画：200ms ease-out
```

### 6. 滑块控件 (Slider)
```
轨道高度：8px
轨道圆角：4px
轨道颜色：#2D2D2D

进度条：
- 颜色：根据数值范围变化
- 最小值：#34C759 (绿)
- 中值：#FFCC00 (黄)
- 最大值：#FF6B35 (橙)

滑块：
- 尺寸：32px 圆形
- 背景：#FFFFFF
- 阴影：0 2px 8px rgba(0,0,0,0.4)

数值显示：
- 位置：滑块上方
- 背景：#1A1A1A 气泡
- 字体：SF Mono, 18px
```

### 7. 输入框 (Input)
```
高度：56px
背景：#1A1A1A
圆角：12px
边框：1px solid #404040

聚焦状态：
- 边框颜色：#007AFF

文字：
- 输入值：18px, #FFFFFF

图标：
- 尺寸：24px

### 8. 导航栏 (Tab Bar)
```

字体：14px, Semibold

- 进入：从顶部滑入 300ms

## 🎬 动效规范

### 微交互时长
| 类型 | 时长 | 缓动函数 |
|------|------|----------|
| 按钮按下 | 100ms | ease-out |
| 页面过渡 | 300ms | ease-in-out |
| 数值变化 | 200ms | ease-out |
| 曲线绘制 | 实时 | linear |

### 特殊动画
- **温度上升**：数字滚动动画，带缓动
- **连接成功**：图标缩放 + 涟漪效果
- **烘焙完成**：庆祝动画 (彩带 + 音效)
- **警告**：红色边框闪烁 + 震动反馈
