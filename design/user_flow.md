## 📱 各步骤详细说明

### 步骤 1: 启动与设备连接

| 元素 | 说明 |
|------|------|
| **界面** | 闪屏 → 设备连接页 |
| **用户操作** | 打开应用，选择并连接烘焙机 |
| **系统反馈** | 扫描动画 → 连接进度 → 连接成功提示 |
| **异常处理** | 未找到设备时提示检查蓝牙/WiFi |
| **时间预估** | 3-10秒 |

**界面元素：**
- 中央：大型咖啡机图标，带脉动动画
- 下方：发现的设备列表（名称 + 信号强度）
- 底部：刷新按钮、手动输入IP按钮

---

### 步骤 2: 主界面 (Dashboard)

| 元素 | 说明 |
|------|------|
| **界面** | 主控制面板 |
| **用户操作** | 查看状态、进入历史、开始新烘焙 |
| **系统反馈** | 实时显示当前温度（若已连接）|
| **快捷入口** | 上次烘焙摘要、快速开始按钮 |

**界面元素：**
- 顶部：设备连接状态 + 电池电量
- 中央：大温度显示（或"未开始"提示）
- 下方：最近3次烘焙记录缩略图
- 底部导航：主页 / 历史 / 设置

---

### 步骤 3: 历史记录界面

| 元素 | 说明 |
|------|------|
| **界面** | 烘焙历史列表 |
| **用户操作** | 浏览、搜索、筛选、查看详情 |
| **系统反馈** | 列表加载、筛选结果更新 |
| **操作** | 点击卡片查看详情，左滑删除 |

**界面元素：**
- 顶部：搜索框 + 筛选按钮
- 列表：烘焙卡片（日期、豆种、曲线缩略图）
- 空状态：提示"还没有烘焙记录"

---

### 步骤 4: 烘焙参数配置

| 元素 | 说明 |
|------|------|
| **界面** | 配置向导 |
| **用户操作** | 选择/创建曲线模板，输入生豆信息 |
| **系统反馈** | 表单验证、自动填充建议 |
| **智能功能** | 根据豆种推荐起始参数 |

**配置项：**

| 配置项 | 输入方式 | 说明 |
|--------|----------|------|
| 曲线模板 | 下拉选择 | 预设模板或历史曲线 |
| 生豆产地 | 文本输入 + 常用选项 | 埃塞俄比亚、哥伦比亚等 |
| 豆种 | 文本输入 + 常用选项 | 耶加雪菲、瑰夏等 |
| 处理方式 | 单选按钮 | 水洗、日晒、蜜处理 |
| 生豆重量 | 数字输入 | 克 (g) |
| 目标烘焙度 | 滑动选择器 | 浅烘 → 深烘 |
| 目标温度 | 数字输入 | 默认根据烘焙度推荐 |
| 备注 | 多行文本 | 可选 |

---

### 步骤 5: 烘焙进行中

| 元素 | 说明 |
|------|------|
| **界面** | 实时监控界面 |
| **用户操作** | 监控数据、调节参数、标记阶段 |
| **系统反馈** | 实时温度更新、曲线绘制、警报提示 |
| **安全功能** | 超温警报、自动切断保护 |

**数据展示：**

| 数据项 | 位置 | 显示格式 |
|--------|------|----------|
| 豆温 | 中央大字体 | 185.5°C |
| 排温 | 左上 | 165.2°C |
| 环境温 | 右上 | 25.3°C |
| ROR | 左下 | +8.5°C/min |
| 时间 | 右下 | 05:32 |
| 风量 | 侧边 | 2800 RPM |
| 火力 | 侧边 | 75% |

**阶段标记按钮：**
- **TP** (Turning Point): 回温点
- **FC** (First Crack): 一爆开始
- **SC** (Second Crack): 二爆开始（可选）
- **Drop**: 下豆/结束

**控制滑块：**
- 火力调节：0-100%，步进5%
- 风量调节：0-3500 RPM，步进100 RPM

---

### 步骤 6: 烘焙完成

| 元素 | 说明 |
|------|------|
| **界面** | 烘焙报告页 |
| **用户操作** | 查看报告、评分、添加备注、保存/分享 |
| **系统反馈** | 数据保存确认、分享菜单 |
| **数据导出** | PDF报告、CSV原始数据、图片分享 |

**报告内容：**

| 项目 | 说明 |
|------|------|
| 烘焙概览 | 总时长、最终温度、失重率估算 |
| 温度曲线 | 完整烘焙曲线图 |
| ROR曲线 | 升温速率变化图 |
| 关键时间点 | TP时间、FC时间、发展时间 |
| 阶段占比 | 脱水期:梅纳期:发展期 比例 |
| 用户评分 | 1-5星评分 |
| 用户备注 | 文本记录 |

```
                    │
        ┌──────────┼──────────┐
        ▼          ▼          ▼
    设备设置    烘焙预设    系统设置
        │          │          │
        ▼          ▼          ▼
    • PID参数   • 新建预设  • 单位切换
    • 传感器校准 • 编辑预设  • 通知设置
    • 安全温度   • 删除预设  • 关于/帮助
```

### 异常处理流程

```
设备断开连接
    │
    ▼
检测信号丢失
    │
    ▼
显示警告横幅 + 震动提醒
    │
    ▼
┌──────────────┐
│ 尝试自动重连 │ ←── 3次尝试
└──────┬───────┘
       │
   ┌───┴───┐
   ▼       ▼
 成功    失败
   │       │
   ▼       ▼
 恢复    提示手动重连
 连接    [重试] [检查设备]
```

## ⏱️ 时间线参考

| 阶段 | 典型时长 | 说明 |
|------|----------|------|
| 启动+连接 | 5-10秒 | 应用启动和设备连接 |
| 参数配置 | 30秒-2分钟 | 输入生豆信息，选择模板 |
| 烘焙过程 | 8-15分钟 | 实际烘焙时间 |
| 保存评价 | 30秒-1分钟 | 查看报告，添加备注 |
| **总计** | **10-18分钟** | 单次完整烘焙流程 |

## 📊 用户决策点

| 决策点 | 选项 | 默认选择 |
|--------|------|----------|
| 连接设备 | 扫描列表 / 手动输入 | 扫描列表 |
| 曲线模板 | 新建 / 使用预设 / 复制历史 | 上次使用的模板 |
| 烘焙控制 | 自动 / 手动调节 | 根据模板自动 |
| 结束方式 | Drop按钮 / 达到目标温度 | 手动Drop |
| 数据分享 | 保存本地 / 导出PDF / 分享图片 | 保存本地 |
